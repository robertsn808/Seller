How to Get Your Credentials
Step 1: Facebook Page Access Token
  1. Go to Facebook Graph Explorer
  2. Select your Facebook App (or create one)
  3. Select your Facebook Page
  4. Add permissions: pages_manage_posts
  5. Generate and copy the access token
Step 2: Facebook Page ID
  1. Go to your Facebook Page
  2. Click "About" in the left sidebar
  3. Scroll down to find "Page ID"
  4. Copy the numeric ID
Alternative: Page ID from URL

Visit facebook.com/your-page-name and look at the page source. Search for "page_id" to find your numeric ID.

Note: The access token needs to be a Page Access Token with posting permissions, not a User Access Token.
